# TrueScan: Breast Cancer Detection with Machine Learning

## Project Overview
TrueScan is a machine learning-based breast cancer detection system designed to classify histopathology images as benign or malignant. The project combines deep learning and traditional machine learning techniques to improve classification accuracy and diagnostic reliability.

## Objective
To develop an intelligent image classification system that assists in early detection of breast cancer using automated feature extraction and predictive modeling.

- Comparative model analysis
- Performance evaluation using accuracy, precision, recall, and F1-score
- Web-based visualization interface

## Outcome
The system successfully classifies breast cancer histopathology images into benign and malignant categories with optimized model performance.
